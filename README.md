# Audio Denoising: Classical vs. Lightweight ML Methods

A comparison of classical signal processing and lightweight machine learning
approaches for speech enhancement, using the VoiceBank+DEMAND dataset.

## Project Overview

This project investigates how classical methods (spectral subtraction, geometric
spectral subtraction) compare to lightweight ML models (MLP, denoising autoencoder)
in terms of denoising performance and computational cost. All methods operate on
spectrogram-based representations of audio.

## Setup

### 1. Clone the repository

```bash
git clone https://github.com/MordoTheHacker/Applied-ML-Audio-Denoising-RUG-2026
cd Applied-ML-Audio-Denoising-RUG-2026
```

### 2. Create and activate a virtual environment

```bash
python -m venv venv

# Mac/Linux
source venv/bin/activate

# Windows
venv\Scripts\activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Download and prepare the dataset

```bash
python src/data.py
```

This will automatically download VoiceBank+DEMAND into `data/raw/` and
prepare wav files in `data/raw/wavs/`.

| Denoising Autoencoder (DAE)    | ML                 |

## Evaluation

Models are evaluated using the standard VoiceBank+DEMAND metrics:

- **PESQ** — Perceptual speech quality (-0.5 to 4.5, higher is better)
- **STOI** — Speech intelligibility (0 to 1, higher is better)
- **CSIG / CBAK / COVL** — Composite quality measures
- **Computational cost** — training time, inference time, model size
